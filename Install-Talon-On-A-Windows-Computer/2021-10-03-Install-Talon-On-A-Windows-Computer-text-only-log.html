Step 3 is to open the Talon Home Directory.

There are many ways of doing this.

On windows my preferred way is to go to the Talon icon, go to scripting, and open ~/Talon.

This opens the Talon home directory.

The next step is to add some scripts to the Talon home directory user folder.

Talon is incredibly customizable.

You can use Talon to set any English voice phrase to any action on your computer.

There are other languages and other input methods that Talon accepts, but since I am an English speaker and I use English words, that's what I need Talon to do for me the most.

As the instructions say, your user scripts control all of the voice commands in Talon. So Talon won't recognize any commands until you add some scripts.

I'm going to the getting scripts section.

There is one script set that is currently recommended to install.

Knauj Talon script set, started by Jeff Knauh, and maintained by about a dozen Talon users.

It is one of the biggest, most robust, and least buggy scripts that's currently available and, by default, it is the recommended script set to install.

To install it I go to the Jeff Knaus Github repository.

I click on the green code button.

I click download zip.

It asks 'What should Firefox do with this file? I wish to save scripts set.

I go to the download button in Firefox...

...and I click open knausj.

I want to move this folder into my Talon home directory user folder.

Click on the file knausj_talon-master and press copy.

I go down to windows file explorer,

find the Talon Home directory,

I click on the user directory.

I right click on the folder anywhere inside it and paste knausj_talon-master to the user directory.

Now I have the scripts I need installed.

Back to Firefox and hit the back button to return to my installation instructions.

To get started learning Talon, go to the side bar on the Talon documentation website and click the link 'Learning Talon.'

The Talon practice website is a good place to go for beginners.

The Talon practice website assumes that you are using the knausj talon repository

If you go through the lessons in order, it will teach you most of the very basics you need to know to get started controlling your computer entirely hands-free.
